# Proyecto INF322: Salas USM

## Instrucciones de uso:
Dentro de carpeta salas-usm-qr:
```bash
cd salas-usm-qr
```
Para ejecutar de manera local, instalamos dependencias:
```bash
npm install
```
Luego, ejecutamos con:
```bash
npm run dev
```
Finalmente, **ctrl** + click en enlace.
